Hello! This is the north dallas astrodynamics research team's research on the efficiency of different muon detection methods (scintillation, geiger counters LiPo etc) on the ground and in the air in a high altitude weather balloon. We started this project since there was a scarce amount of information on muon detectors when I wanted to learn about them. In all, this project aims to add more clarity in the world of muon detectors, scintillators, best material types, and the strengths and weaknesses of the detectors we test.
